# Лабораторная работа №4 — Юзабилити-тестирование

Учебный проект для автоматизированного юзабилити-тестирования интернет-ресурса с использованием **Яндекс.Метрики**.

Проект выполнен в формате **одного HTML-файла**: весь HTML, CSS и JavaScript находятся в `index.html`.

## Цель работы

Провести автоматизированное юзабилити-тестирование и построить тепловую карту интернет-ресурса.

## Используемое средство

В качестве средства юзабилити-тестирования используется **Яндекс.Метрика**.

В код страницы подключён счётчик Яндекс.Метрики с включёнными параметрами:

- `webvisor: true` — запись действий пользователя;
- `clickmap: true` — карта кликов;
- `trackLinks: true` — отслеживание переходов по ссылкам;
- `accurateTrackBounce: true` — уточнённый показатель отказов.

## Что реализовано на странице

- тестовая форма авторизации;
- поле логина или email;
- поле пароля;
- кнопка показа и скрытия пароля;
- проверка пустых полей;
- проверка неверного логина;
- проверка неверного пароля;
- сообщение об успешном входе;
- подключение Яндекс.Метрики для сбора кликов и просмотра карты кликов.

## Тестовые данные

Для успешного входа можно использовать один из вариантов:

```text
testuser / password123
admin / admin123
ivan@example.com / password123
```

## Как запустить

Самый простой способ:

1. Скачать репозиторий.
2. Открыть файл `index.html` в браузере.
3. Выполнить тестовые действия на форме авторизации.

Для полноценного просмотра карты кликов страницу нужно разместить на хостинге, например через GitHub Pages, потому что Яндекс.Метрика должна собирать данные с опубликованной страницы.

## Задание для тестирования

Пользователю предлагается выполнить следующие действия:

1. Открыть страницу авторизации.
2. Попробовать нажать кнопку «Войти» с пустыми полями.
3. Ввести неверный пароль и проверить сообщение об ошибке.
4. Ввести корректные данные и выполнить успешный вход.
5. Проверить работу кнопки показа и скрытия пароля.

## Как посмотреть тепловую карту

1. Опубликовать сайт через GitHub Pages.
2. Открыть опубликованную страницу в браузере.
3. Выполнить несколько кликов по форме авторизации.
4. Открыть Яндекс.Метрику.
5. Выбрать счётчик сайта.
6. Перейти в раздел с картами и открыть карту кликов.
