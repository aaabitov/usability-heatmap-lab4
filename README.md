# Лабораторная работа №4 — Юзабилити-тестирование

Учебный проект для автоматизированного юзабилити-тестирования интернет-ресурса и построения тепловой карты кликов.

Проект выполнен в формате **одного HTML-файла**: весь HTML, CSS и JavaScript находятся в `index.html`.

## Цель работы

Провести автоматизированное юзабилити-тестирование и построить тепловую карту интернет-ресурса.

## Что реализовано

- интерактивный макет интернет-ресурса;
- запись координат кликов пользователя;
- построение тепловой карты на `canvas`;
- сохранение данных в `localStorage`;
- загрузка демонстрационных кликов;
- очистка результатов тестирования;
- просмотр результатов в форматах JSON и CSV;
- расчёт простых показателей: количество кликов, активные зоны, последний клик.

## Как запустить

Самый простой способ:

1. Скачать репозиторий.
2. Открыть файл `index.html` в браузере.
3. Нажимать на элементы страницы.
4. Смотреть, как формируется тепловая карта кликов.

Также можно запустить проект через локальный сервер:

```bash
python -m http.server 8000
```

После запуска открыть в браузере:

```text
http://localhost:8000
```

## Задание для тестирования

Пользователю предлагается выполнить следующие действия:

1. Найти кнопку создания турнира.
2. Посмотреть возможности сайта.
3. Заполнить форму заявки.

После выполнения действий программа показывает, какие зоны страницы чаще всего использовались пользователем.

## Как работает тепловая карта

При каждом клике программа определяет координаты нажатия относительно тестируемой области страницы. Эти координаты сохраняются в браузере с помощью `localStorage`. Затем на элементе `canvas` рисуются цветные круги: чем больше кликов в одной зоне, тем заметнее становится участок карты.

Проект подготовлен для лабораторной работы №4 по теме «Юзабилити-тестирование». Он демонстрирует принцип сбора действий пользователя и визуализации активных зон интерфейса в виде тепловой карты.
