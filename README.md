# Лабораторная работа №4 — Юзабилити-тестирование

Веб-приложение для учебного автоматизированного юзабилити-тестирования и построения тепловой карты кликов.

## Что реализовано

- интерактивный макет интернет-ресурса;
- запись координат кликов пользователя;
- построение тепловой карты на canvas;
- сохранение данных в localStorage;
- загрузка демонстрационных кликов;
- очистка результатов тестирования;
- экспорт данных в JSON и CSV;
- расчёт простых показателей: количество кликов, активные зоны, последний клик.

## Как запустить

Откройте файл `index.html` в браузере.

Также можно запустить локальный сервер:

```bash
python -m http.server 8000
```

После запуска откройте в браузере:

```text
http://localhost:8000
```
